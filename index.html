    사용자 메뉴얼

    📖 사용 가이드

    1. 환경 설정

    • 접속 브라우저 선택
    - 안드로이드폰 : 구글앱
    - 아이폰 : Bluefy 앱

    • Gemini AI, STRAVA 페어링
    - 설정 : My CAREER > STELVIO AI
    - Gemini AI : 훈련 분석 및 개인 맞춤 워크아웃 작성, 추천
    - STRAVA : 인도어와 아웃도어 모든 훈련 로그 수집

    2. 기본 사용법

    • INDOOR TRAINING: 그룹 실내 훈련에 참여할 수 있습니다.
    • SOLO TRAINING: 개인 훈련을 시작할 수 있습니다.
    • MY CAREER: 워크아웃, 훈련 일지, 사용자 정보, 환경설정 등을 관리합니다.

    3. MY CAREER 메뉴

    • 훈련 스케줄: 주간/월간 훈련 계획을 수립하고 관리합니다.
    • 훈련 일지: 훈련 기록과 성과를 확인합니다.
    • 워크아웃: 훈련 프로그램을 선택하거나 생성합니다.
    • 사용자 정보: 프로필과 사용자 대시보드를 관리합니다.
    - 사용자 대시보드: 내 훈련 기록과 컨디션을 AI가 분석하여, 오늘의 최적화된 훈련을 추천합니다.

    4. 훈련 시작하기

    • BLE 센서(파워미터, 심박계, 스마트로라)를 연결하세요.
    • 훈련할 워크아웃을 선택하세요.
    • 훈련 중 실시간 데이터를 확인하며 목표를 달성하세요.

    5. 아이폰(iOS) 사용자 : 블루투스 연동 가이드
    Step 1. 전용 앱 설치

    앱스토어에서 'Bluefy' 검색 후 설치
    앱 이름: Bluefy - Web BLE Browser

    Step 2. 권한 허용 (필수)

    앱 실행 시 블루투스 사용 권한 팝업에서
    반드시 [허용] 또는 [확인]을 눌러주세요.

    Step 3. STELVIO AI 접속

    Bluefy 앱 주소창에 stelvio.ai.kr 입력
    💡 Tip: 하단 메뉴의 '북마크(별 모양)'를 활용하면 편리합니다.

    Step 4. 센서 연결

    화면 내 [장비 연결] 클릭
    → 목록에서 센서 선택
    → [Pair] 클릭

    6. STRAVA 로그 수집 불가 해결 가이드

    (스트라바와 연결은 되었으나 로그 데이터가 안 불러와지는 경우)

    • 1) 과거 활동 일괄 공개 전환 (가장 중요)
    - 스트라바 앱 실행 ➡️ 우측 상단 톱니바퀴(설정) ➡️ [프라이버시 컨트롤 (Privacy Controls)] ➡️ 하단의 [과거 활동 수정 (Edit Past Activities)] 클릭
    - '활동 공개 대상(Activity Visibility)'을 선택하고 [모두(Everyone)] 로 변경하여 일괄 적용해 주세요.

    • 2) STELVIO 앱 접근 권한 재인증
    - 스트라바 앱 설정 ➡️ [다른 앱 (Partner Integrations / Apps)] ➡️ STELVIO 앱 선택 후 [연결 해제(Disconnect)]
    - 그 후 STELVIO 대시보드로 돌아와 다시 스트라바 연동을 진행해 주세요.
    - 💡 주의: 연동 시 뜨는 스트라바 인증 화면에서 "비공개 활동에 대한 데이터 보기" 체크박스를 절대 해제하지 말고 모두 동의해 주셔야 전체 로그 분석이 가능합니다.

    🚴‍♂️ 스트라바(Strava) 신규 가입 가이드

    현재 스트라바의 한국 스토어 서비스 제한으로 인해 일반적인 앱 검색 및 신규 가입이 어려운 상태입니다. STELVIO AI의 원활한 훈련 기록 연동을 위해, 아래의 우회 가입 및 설치 방법을 단계별로 안내해 드립니다. 천천히 따라 해 주시면 정상적으로 훈련을 시작하실 수 있습니다.

    1. [공통] 스트라바 웹사이트 회원가입 (VPN 필요)

    한국 IP로는 가입 접근이 차단될 수 있으므로, 최초 1회 가입 시에만 VPN(가상사설망) 우회가 필요합니다.

    • 추천 무료 VPN (PC/모바일 모두 사용 가능):
      • ProtonVPN (광고가 없고 안전한 무료 버전 제공)
      • TunnelBear
    • 가입 절차:
      1. VPN 프로그램을 설치하고 실행하여 접속 위치를 미국 또는 일본으로 설정합니다.
      2. 스트라바 공식 웹사이트(strava.com)에 접속합니다.
      3. 이메일을 사용하여 신규 가입을 진행합니다. (프로필상의 거주 국가도 우회한 국가로 동일하게 설정해 주세요.)
      4. 가입이 완료되면 VPN은 바로 종료하셔도 됩니다.

    2. [안드로이드 기기] 스트라바 앱 설치

    안드로이드 사용자는 구글 플레이스토어 대신, 안전성이 검증된 해외 스토어에서 설치 파일(APK)을 직접 다운로드해야 합니다.

    • 다운로드 링크 (APKMirror 공식 페이지):
      https://www.apkmirror.com/apk/strava-inc/strava-running-and-cycling-gps/
    • 설치 절차:
      1. 스마트폰 브라우저에서 위 링크에 접속합니다.
      2. 목록에서 가장 위에 있는 최신 버전(Latest Version)의 우측 다운로드 아이콘(⬇️)을 누릅니다.
      3. 다음 화면에서 'Download APK' 버튼을 눌러 파일을 스마트폰에 저장합니다.
      4. 다운로드된 파일을 실행합니다. '출처를 알 수 없는 앱 설치' 경고가 나타나면 허용을 눌러 기기에 설치를 완료해 주세요.

    3. [아이폰 / iOS] 스트라바 앱 설치

    아이폰은 외부 파일 직접 설치가 불가능하므로, 앱스토어의 국가 설정을 잠시 해외로 변경하여 다운로드해야 합니다.

    • 설치 절차:
      1. 아이폰의 App Store 앱을 엽니다.
      2. 우측 상단의 내 프로필 사진을 누른 후, 상단의 내 계정(이름/이메일)을 한 번 더 누릅니다.
      3. '국가/지역' ➡️ '국가 또는 지역 변경'을 선택합니다.
      4. 국가를 미국(United States)으로 변경합니다. (결제 수단은 'None'으로 체크하시고, 주소는 구글맵 등에서 임의의 미국 주소나 배송대행지 주소를 입력하시면 됩니다.)
      5. 앱스토어 검색창에 'Strava'를 검색하여 앱을 다운로드합니다.
      6. 설치가 완료되면 다시 동일한 경로로 들어가 국가를 '대한민국'으로 원상 복구하시면 평소처럼 사용 가능합니다.
